Pipetally Analysis

Ficus' PAAT software performs automatic pipe tally analysis using multiple levels of analysis, including level 1, level 2, finite element, reliability, and defect matching and automatic report generation.

Optimize maintenance plans

Find an optimum Maintenance Planning and Scheduling for the corroded pipeline. The Maintenance Report provide critical defects, their time to fail and sugestion on sections to repair. Future Analysis is performed based on the CGR given or computed based on 2 or more successive pig inspection (pipe tally files) or based on the pipe installation date. Defect Matching needed is effectively performed using an algorithm based on the longest common subsequence (LCS) problem.

Focus your time where it really matters

Using the PAAT, the results are computed and generated in real time, freeing up time from repetitive tasks so that engineers can perform more critical and specific tasks.
